## Switcher (tool)

From the **repository root**:

```bash
./docs/switch_readme.sh zh
```

Or:

```bash
python3 docs/switch_readme.py zh
```

Valid codes: `en`, `zh`, `ja`, `ko`, `es`.

This overwrites the root `README.md` with the chosen translation and rewrites language links so they still work from the repo root (e.g. `docs/README.ja.md`).

To read a locale **without** changing the root file, open the corresponding file in this folder on GitHub or locally.

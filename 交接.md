# YXL-LACE（Apple Silicon / M 芯片）打包 arm64 DMG 交接文档

> 目标：在 **M1/M2/M3…（arm64）Mac** 上，从源码打包出可分发的  
> `dist/YXL-LACE-GUI-arm64.dmg`（未签名版）。

---

## 0) 前置检查（必须）

打开终端执行：

```bash
uname -m
```

输出必须是：

- `arm64`

---

## 1) 获取代码

任选一种方式：

### A. 用 git

```bash
git clone <仓库地址>
cd YXL-LACE
git checkout <分支名或提交号>
```

### B. 下载 zip

解压后 `cd` 到项目根目录（里面要有 `src/`、`requirements.txt`）。

---

## 2) 创建并激活 venv（Python 3.10+）

```bash
python3 --version
python3 -m venv .venv
source .venv/bin/activate
```

（可选但推荐）清理代理环境变量，避免 pip 报 SOCKS 依赖问题：

```bash
for _v in ALL_PROXY all_proxy HTTP_PROXY HTTPS_PROXY http_proxy https_proxy; do
  unset "${_v}" 2>/dev/null || true
done
```

---

## 3) 安装依赖（必须）

```bash
pip install -r requirements.txt
pip install -r requirements-gui.txt
pip install pyinstaller
```

说明：
- `requirements.txt`：核心加密依赖（cryptography）
- `requirements-gui.txt`：GUI 依赖（PySide6）
- `pyinstaller`：打包工具

---

## 4) 打包前先跑一次 GUI（确认能起）

```bash
PYTHONPATH=src python -m yxl_lace.ui_qt
```

能弹窗再继续。

---

## 5) 生成 `.app`（arm64）

在项目根目录执行：

```bash
PYTHONPATH=src python -m PyInstaller --noconfirm \
  --windowed \
  --name "YXL-LACE-GUI" \
  --paths src \
  --icon assets/app.icns \
  --add-data "assets/app.png:assets" \
  run_ui.py
```

产物：
- `dist/YXL-LACE-GUI.app`

注意：
- 一定要用 **venv 内**的 `python -m PyInstaller`
- 一定要带 `--paths src`，否则可能找不到 `yxl_lace`

---

## 6) 打包成 `.dmg`

```bash
rm -rf dist/dmg_stage
mkdir -p dist/dmg_stage
cp -R dist/YXL-LACE-GUI.app dist/dmg_stage/
ln -s /Applications dist/dmg_stage/Applications

hdiutil create -volname "YXL-LACE-GUI" \
  -srcfolder dist/dmg_stage \
  -ov -format UDZO \
  dist/YXL-LACE-GUI-arm64.dmg
```

产物：
- `dist/YXL-LACE-GUI-arm64.dmg`

---

## 7) 自测（推荐）

```bash
open dist/YXL-LACE-GUI-arm64.dmg
```

把 App 拖到 Applications 后双击运行。

### 若被 macOS 拦截（未签名版正常）

- 系统设置 → 隐私与安全 → 找到拦截提示 → 点“仍要打开”

或（更直接）：

```bash
xattr -dr com.apple.quarantine "/Applications/YXL-LACE-GUI.app"
```
